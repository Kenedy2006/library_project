# Library Management System

Proyecto completo con:
- Backend en Django (API REST)
- Frontend en React

## Requisitos
- Python 3.8+
- Node.js 14+

## Instalación

### Backend
```bash
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## Estructura
```
backend/       # API Django
frontend/      # Aplicación React
```
